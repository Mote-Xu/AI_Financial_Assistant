## 项目概况

AI 财务助手是一个**个人理财分析系统**，运行在本地 Windows 11 上。目标是让 AI 基于用户的个人财务数据（资产、收入、保单、负债、目标）和市场实时行情，提供个性化的财务建议和分析报告。

## 技术架构

三层分离：

1. **数据层** — `finance/` 目录下的 Markdown 文件，手动维护个人财务数据
2. **数据接入层** — `scripts/market_data.py`，使用 `akshare`（免费 Python 库）获取 A 股 ETF 实时价和基金净值
3. **分析层** — `scripts/deepseek_analysis.py`，将本地数据 + 市场行情组合后，通过 DeepSeek API 生成分析报告

交互由 Claude Code CLI 完成——用户可以自然语言下达指令，Claude Code 调度脚本执行。

## 待解决问题

1. **编码问题**：Git Bash 下 `conda run` 输出中文乱码（UTF-8 → GBK 转换失败），文件写入正常

4. **隐私权衡**：分析时财务数据上传 DeepSeek 云端，尚未评估风险
