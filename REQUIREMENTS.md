# 功能需求清单

> 最后更新：2026-06-26

---

## P0 — 核心功能

- [x] **市场数据获取**：A 股 ETF（eastmoney）+ 个股（eastmoney/Sina 双源）+ 基金净值 ✅
- [x] **持仓市值计算**：基于当前行情自动更新资产总值 ✅
- [x] **资产配置分析**：输出各类资产占比，对比目标配置 ✅
- [x] **现金流分析**：月收入/支出/结余、应急储备覆盖月数 ✅
- [x] **保险保障缺口**：基于年龄/收入/负债评估保险是否充足 ✅

## P1 — 重要功能

- [x] **投资组合再平衡建议**：偏离度检测 + 具体调仓方案 + 金额 ✅
- [x] **财务目标进度追踪**：各目标完成百分比、预计达成时间 ✅
- [x] **市场事件应急分析**：重大行情变化时的影响评估和应对 ✅
- [x] **月度自动体检报告**：一键生成月度财务健康报告 + 推送 ✅

## P2 — 增强功能

- [x] **推送通知**：企业微信机器人（完整报告多段拆分）+ Server酱（兜底）✅
- [ ] **定时自动执行**：通过 cron/计划任务定时拉数据 + 分析 + 推送
- [ ] **历史净值追踪**：每次运行存档，画资产曲线
- [ ] **市场波动预警**：持仓单日跌超阈值自动推送
- [ ] **FIRE 模拟器**：输入参数模拟不同路径下达标时间
- [ ] **定投回测**：历史数据回测定投策略效果
- [ ] **多账户汇总**：股票/基金/银行账户统一视图

## 非功能需求

- [x] 所有真实财务数据不入 git ✅
- [x] 分析脚本支持脱敏数据运行 ✅
- [x] 错误处理健壮（API 失败、网络不可用、代理拦截） ✅
- [x] 分析输出可读性（中文、分层、具体数字） ✅
