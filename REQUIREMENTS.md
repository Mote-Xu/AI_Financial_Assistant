# 功能需求清单

> 最后更新：2026-06-25

---

## P0 — 核心功能

- [ ] **市场数据获取**：自动拉取 A 股 ETF 实时价、基金净值
- [ ] **持仓市值计算**：基于当前行情自动更新资产总值
- [ ] **资产配置分析**：输出各类资产占比，对比目标配置
- [ ] **现金流分析**：月收入/支出/结余、应急储备覆盖月数
- [ ] **保险保障缺口**：基于年龄/收入/负债评估保险是否充足

## P1 — 重要功能

- [ ] **投资组合再平衡建议**：偏离度检测 + 具体调仓方案
- [ ] **财务目标进度追踪**：各目标完成百分比、预计达成时间
- [ ] **市场事件应急分析**：重大行情变化时的影响评估和应对
- [ ] **月度自动体检报告**：一键生成月度财务健康报告

## P2 — 增强功能

- [ ] **定时提醒**：通过 cron 定时拉数据 + 推送提醒
- [ ] **多账户汇总**：支持股票账户、基金账户、银行账户统一视图
- [ ] **定投模拟**：基于历史数据回测定投策略效果
- [ ] **税务估算**：A 股印花税、基金赎回费计算

## 非功能需求

- [ ] 所有真实财务数据不入 git
- [ ] 分析脚本支持脱敏数据运行（开发和测试阶段）
- [ ] 错误处理健壮（API 调用失败、网络不可用等）
- [ ] 分析输出可读性（中文、分层次、有具体数字）
